# 小红书评论区智能分析系统

一个功能强大的小红书评论区智能总结和检索系统。

## 功能特性

- 📊 **智能总结**: 自动分析评论情感，统计正面、负面、中性评价
- 🔍 **智能检索**: 根据关键词快速搜索相关评论
- 🎯 **情感分析**: 基于关键词的情感倾向性分析
- 🏷️ **关键词提取**: 自动提取评论热门关键词
- 🔗 **一键跳转**: 支持一键定位并复制评论链接
- 📝 **手动导入**: 支持手动粘贴小红书评论数据进行分析
- 🗑️ **数据管理**: 支持清空并恢复示例数据

## 技术栈

- **后端**: Python Flask
- **前端**: HTML, CSS, JavaScript
- **NLP**: jieba 分词

## 快速开始

1. 安装依赖:
```bash
pip install -r requirements.txt
```

2. 运行应用:
```bash
python app.py
```

3. 打开浏览器访问: http://localhost:5000

## 使用说明

### 导入小红书评论

1. 打开小红书笔记，复制评论区内容
2. 点击页面上的 **"📝 导入评论"** 按钮
3. 在弹窗中粘贴评论，每行一条，格式如下：
   ```
   用户名|评论内容|点赞数
   ```
   例如：
   ```
   小明|这个产品真好用！|234
   小红|效果不太明显|45
   ```
4. 点击"确认导入"即可分析

### 搜索功能

在搜索框输入关键词（如"敏感肌"、"价格"），点击搜索即可找到相关评论，并查看搜索结果的独立分析。

### 一键跳转

点击任意评论下方的 **"🔗 复制链接"** 按钮，页面会自动滚动到该评论位置并高亮显示，同时评论链接会被复制到剪贴板。

## API 接口

- `GET /` - 主页
- `GET /api/comments` - 获取所有评论和总结
- `GET /api/search?q=关键词` - 搜索评论
- `POST /api/import` - 导入评论数据
- `POST /api/clear` - 清空并恢复示例数据
